About

Feature Creep Robot comic was created by me, Fernando Bevilacqua, a Computer Science professor and game developer. The main idea is to expose and poke fun of the funny situations that happen in the game development world.

If you lived or witnessed any funny moment while working with games, please drop me a note so we can all laugh about it.
